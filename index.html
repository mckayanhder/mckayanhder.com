A few of my favorite projects. Feel free to click on the links to see them live.

Artraction - This is a graphing equation art generator that creates unique artwork, all from using the "Peter De Jong Attractor" and a canvasing javascript library. Built with HTML/CSS/JS and p5.js library. This project was a way for me to learn more into math dense programming as well as dealing with challenges with design from a relatively old and un-customizable library. Taking input from the user by adjusting sliders you can change the equation values as well as RGB and Opacity.

Chatter.io - A business facing chat application with CRUD capabilities. Built with HTML/CSS/REACT - and utilizing a new backend server chatengine.io. Building this app helped me gain experience using new third party technologies as well as server/client relationships. The backend server generates a unique key and portal to add members with a verified log in and admin privileges.
