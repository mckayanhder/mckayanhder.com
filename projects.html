a small handful of my favorites

Artraction is a digital art generator using a graphing equation that makes beautiful images. I utilized a p5.js to sketch the image out. This project includes a source link to the reading material for the equation. An interesting read..Give it a try.

Chatter is a team focused messaging app that can CRUD through creating chatrooms. Utilizing Chatengine.io for the back end hosting as well as all server side utilities like creating users, editing rooms, adding webhooks. Give it a try.

Neumarket is a React API project cryptocurrency tracker that allows you to search through the top traded and valued currencies using coinpaprika api. I had originally used a different one but wanted one with more data to show. Give it a try.

The first app I ever downloaded on a smart device was a piano sound generator. I used to spend hours working on soundboards making my own and sharing with friends so this was one of my first projects.. would like to update it soon when I get more time. Give it a try.
